## 📊 项目核心数据模型

### 主要业务实体

- 👤 **用户 (User)** - 微信授权用户、个人档案
- 📅 **断食计划 (FastingPlan)** - 168断食配置
- 📝 **断食记录 (FastingRecord)** - 每日断食执行情况
- ⚖️ **体重记录 (WeightRecord)** - 体重变化追踪
- 🏆 **成就系统 (Achievement)** - 激励机制
- 💕 **情侣关系 (CoupleRelation)** - 用户互动
- 🍽️ **饮食记录 (FoodRecord)** - 饮食管理
- 🔔 **消息通知 (Notification)** - 推送提醒

### 数据库技术栈

- **数据库**: MySQL 8.0+
- **ORM**: Prisma
- **缓存**: Redis
- **文件存储**: 腾讯云COS
